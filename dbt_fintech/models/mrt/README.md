# MRT Folder Organization

Current active reporting models are in `mrt/rp/`.

Planned organization as MRT scope grows:
- `mrt/mrt_report/finance/` for finance reporting views
- `mrt/mrt_report/project/` for project/ops reporting views
- `mrt/mrt_cube/` reserved for semantic analytics-serving models later

This folder structure is organizational only at this stage. Existing active models remain unchanged.
