# MART Folder Organization

Current active reporting models are in `mart/rp/`.

Planned organization as MART scope grows:
- `mart/mart_report/finance/` for finance reporting views
- `mart/mart_report/project/` for project/ops reporting views
- `mart/mart_cube/` reserved for semantic analytics-serving models later

This folder structure is organizational only at this stage. Existing active models remain unchanged.
