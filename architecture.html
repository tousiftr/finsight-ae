System flow

Generate → Load → Model → Activate.

FinSight simulates a fintech product analytics and finance reporting platform from synthetic source events through BI and behavioral analytics surfaces.

  • 1. Synthetic sourcesPython generates customers, accounts, transactions, product events, KYC, campaign, risk, and market data.
  • 2. Raw ingestionScheduled GitHub Actions load generated data into raw storage and Neon Postgres.
  • 3. dbt transformationStaging, intermediate, marts, snapshots, seeds, tests, and docs create trusted analytics models.
  • 4. Analytics activationSuperset serves public KPI reporting, while Mixpanel supports product behavior analysis.
  • 5. OperationsGitHub Actions, Airflow, and Dagster demonstrate scheduling, lineage, and observability patterns.
